project outline

pfc_marker is a small windows tool to import standard cmx3600 edls, csv-textfiles, avid mediacomposer markers or dvs clipster timeline markers into an existing pfclean sequence.

it works basically from left to right. buttons and selections are only activated when it makes a logical sense. all timecode calculations are defined by the selected pfclean-sequence.

edl (cmx3600): source- or record-tc only, dissolves are filtered, tapenames mapped as notes
dvs clipster project: timeline-markers including names and comments
csv-text: one marker-event per line, comments are separated by the first comma
avid mediacomposer: mediacomposer markers exported as txt, last number shows markerrange

compiled installer (msi) under /msi, current build under /build runs straight away
written in python3.4 and qt5, tested with the pixelfarm pfclean v.2014-2015 on windows 7/8/10
